# TaskSync

Sistema de gerenciamento de tarefas em PHP + MySQL.

## Como executar

1. Coloque a pasta no htdocs do XAMPP
2. Inicie Apache e MySQL
3. Importe o arquivo database/tasksync.sql no phpMyAdmin
4. Acesse:
http://localhost/tasksync

## Tecnologias
- PHP
- MySQL
- HTML
- CSS

## Funcionalidades
- Cadastro
- Login
- CRUD de tarefas
- Kanban
- Controle de permissões
